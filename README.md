## Prerequisites
* Android Studio
* Android Emulator

## Unit Tests
1. Run `./gradlew  testDebugUnitTest`

## Integration Tests
1. Run `./gradlew connectedDebugAndroidTest`

## Both Tests & Coverage
1. Run `./gradlew clean jacocoTestReport`
